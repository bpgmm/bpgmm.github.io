# Result List

Once the user has performed any type of query (see sections Simple Search, Advanced Search, Batch Search, Batch Advanced Search), a list of results is returned by CMM.

  1. Compound Id: CMM Id.
  2. Name: Name of the putative annotation compound.
  3. Formula: Formula of the putative annotation compound.
  4. Molecular weight: Molecular weight of the putative annotation compound.
  5. Retention time: Retention time introduced by the user for the experimental mass (see [18]).
  6. Error PPM: Difference in parts per million (ppm or mDa) between the molecular weight and the corresponding experimental mass ([18]) and it corresponding adduct ([19]).
  7. Score 1: Score for ionization rules (see χ1 in Annotation rules). The code colour is structured in four ranges:
  • [0, 0.5) is red and means that this annotation is very likely wrong.
  • [0.5, 1) is orange and means that this annotation is likely wrong.
  • [1, 1.5) is yellow and means that this annotation is likely right.
  • [1.5, 2] is green and means that this annotation is very likely right.
  1. Score 2: Score for adduct formation rules (see χ2 in Annotation rules). The code colour is the same than for score 1.
  2. Score 3: Score for lipid elution order (see χ3 in Annotation rules). The code colour is the same than for score 1.
  3. Final score: Integrated score (see χ in Annotation rules). The code colour is the same than for score 1.
  4. Cas: CAS Id.
  5. KEGG Id: KEGG ID and its corresponding link.
  6. HMDB Id: HMDB ID and its corresponding link.
  7. LipidMaps Id: LipidMaps ID and its corresponding link.
  8. Metlin Id: Metlin ID and its corresponding link.
  9. PubChem Id: Pub Chemical Id and its corresponding link.
  10. Pathways: Pathways from KEGG where the compound is present and its corresponding link.
  11. Experimental mass: Experimental mass introduced by the user.
  12. Adduct: Corresponding adduct for this table.
  13. Number of hits: Number of hits found for the search corresponding to experimental mass ([18]) and its corresponding adduct ([19]).
  14. Generate Excel: Button which generates an Excel file with the complete result list (all experimental masses and adducts). This excel file contains the same fields that the on-line interface and the same code colour explained in [7].
